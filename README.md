# World Population Exploratory Data Analysis (EDA)

A comprehensive data analysis project exploring global demographic patterns, growth trajectories across continents, 
and a deep-dive case study on Thailand's demographic transition using historical records from 1970 to 2022.

---

## Project Objectives

* **Top 10 Most Populous Countries:** Identify and visualize the world's most populous nations as of 2022.
* **Correlation Analysis:** Evaluate linear relationships across different decades to study population continuity.
* **Continent-Wise Analysis:** Group regional data to track average growth patterns and identify distinct regional trajectories.
* **Thailand Deep Dive:** Slice and transpose historical data to deeply analyze the population growth trend of Thailand.

---

## Tech Stack

* **Language:** Python
* **Data Manipulation:** `Pandas`
* **Data Visualization:** `Seaborn`, `Matplotlib.pyplot`
* **Environment:** Jupyter Notebook
